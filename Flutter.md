# Flutter

## 在 macOS 上安装和配置 Flutter 开发环境

1. 获取 Flutter SDK
2. 运行 flutter doctor 命令
3. 更新 PATH 环境变量

### 设置 IOS 开发环境

- 安装 Xcode
- 配置 iOS 模拟器(如果 Xcode 安装失败，先下载安装包，用 ∑ 终端命令手动安装)

### 设置 Android 开发环境

- 安装 Android Studio
- 配置 Android 模拟器
- 同意 Android 协议

## 构建您的第一个 Flutter 应用

### 提取 widget

要改为更复杂的设计，一种行之有效的方式是将此代码行提取到单独的 widget 中。为 UI 的单独逻辑部分使用单独的 widget 是在 Flutter 中管理复杂性的一种重要方法。
